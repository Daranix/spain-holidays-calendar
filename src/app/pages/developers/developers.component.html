Para Desarrolladores

CalendarioVacaciones.com es un proyecto de código abierto. Si eres desarrollador y necesitas datos sobre los festivos laborales en España, puedes utilizar nuestra base de datos.

API / Repositorio GitHub

Todos los datos de festivos nacionales, autonómicos y locales se mantienen en nuestro repositorio de GitHub. Puedes acceder a los archivos JSON en crudo o contribuir añadiendo los festivos locales de tu municipio si no están presentes.

Ver Repositorio en GitHub

Formato JSON de Festivos

Los datos se estructuran de manera sencilla para que puedas integrarlos rápidamente en tus aplicaciones.

{{ '{' }}
  "provincia": "madrid",
  "year": 2025,
  "festivos": [
    {{ '{' }}
      "fecha": "2025-01-01",
      "nombre": "Año Nuevo",
      "tipo": "nacional"
    {{ '}' }},
    ...
  ]
{{ '}' }}
